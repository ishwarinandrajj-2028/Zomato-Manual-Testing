# Zomato Website Manual Testing Project

This project demonstrates manual testing of a real-world food delivery web application.

## Objective
To perform functional testing and validation of key features of the Zomato website such as login validation, navigation, and search functionality.

## Testing Type
Manual Testing

## Test Scenarios Covered
- Website loading and accessibility
- Location detection
- Restaurant search functionality
- Restaurant page navigation
- Menu viewing
- Login popup validation
- Phone number input validation
- Navigation between sections

## Files Included
- TestCases_Zomato_ManualTesting.xlsx → Contains detailed test cases and expected results
- BugReport_Zomato.docx → Documents identified usability observation

## Skills Demonstrated
- Manual Testing
- Test Case Design
- Bug Reporting
- Functional Testing
